How I Design Now

The shift since 2025
Before — Pre-2025
01
Research (weeks)
Interviews, synthesis, affinity diagrams. Weeks of work before a single pixel.
02
Wireframes → Reviews → Revisions
Lo-fi to hi-fi across multiple review cycles. Stakeholder feedback loops stretched timelines.
03
Prototype in Figma
Click-through prototypes. Limited to what Figma could simulate, rarely felt real.
04
Handoff → Wait
Handoff doc, then wait for engineering. Lost context, implementation gaps, back-and-forth.
→
Months per feature
Now — AI-Augmented
01
Research + AI synthesis (hours)
Use Claude to analyze data, surface patterns, and generate research frameworks — in hours, not weeks.
02
Explore 10× more directions
AI-assisted ideation means exploring 10 distinct concepts in the time it used to take to sketch 2.
03
Interactive prototype, same day
Build working prototypes in code with Cursor + Claude. Real interactions, real data, same day.
04
Design and ship together
I can close the loop myself — design, build, deploy. No handoff gap. Faster learning cycles.
→
Days per feature
